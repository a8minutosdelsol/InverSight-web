# InverSight Website

This repository contains the public landing page for InverSight.

**InverSight** is a physics-inspired forecasting platform for finance. It helps investment and finance teams transform historical financial metrics into probability ranges, downside-risk indicators, and valuation-sensitive scenarios.

## Website

The site is deployed with GitHub Pages and connected to the domain:

`https://inversight.cl`

## Structure

* `index.html` — main landing page
* `styles.css` — visual styles
* `assets/` — images, logo, and other static files
* `CNAME` — custom domain configuration for GitHub Pages
